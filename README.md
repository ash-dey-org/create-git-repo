Use Terraform to create github repository
=========================================

Creates Gitub repository\
Assigns permission\
Sets default branch\
Sets branch protection rules\
Sets environments for Github Actions

Define the following environment variables

### Terraform cloud

export TF_CLOUD_ORGANIZATION=\
export TFE_TOKEN=\
export TF_WORKSPACE=

### Github
export GITHUB_TOKEN=\
export GITHUB_OWNER=
